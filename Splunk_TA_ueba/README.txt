The Splunk Add-on for UEBA (User and Entity Behavior Analytics) allows a Splunk software administrator to forward events to Splunk UBA as well as index events exported from Splunk UBA.

Documentation for this add-on is located at: http://docs.splunk.com/Documentation/AddOns/latest/UEBA/About

For installation and set-up instructions, refer to the Installation and Configuration section: http://docs.splunk.com/Documentation/AddOns/latest/UEBA/Hardwareandsoftwarerequirements

For release notes, refer to the Release notes section: http://docs.splunk.com/Documentation/AddOns/latest/UEBA/Releasenotes
